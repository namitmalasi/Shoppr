# Shoppr

Shoppr is an e-commerce application, where you can :

- search a product through search bar.
- look at all the products and also a single product.
- make payment through PayPal/credit card.
- adding products to cart.

# Tech Stack

- Reactjs
- Express
- Nodejs
- MongoDB
- Redux
